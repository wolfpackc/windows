# WIN32 API - CONCEPTOS BÁSICOS 1
La arquitectura de fundamenta en programación orientada a eventos, donde el codigo que se crea(la mayoria), esta en estado muerto, a la espera de que un evento de windows lo despierte.


Hechando un primer vistazo acerca de código de win32 te percatas de que existen dos partes importantes. La primera consiste en la gestion de eventos y la otra en la creación de ventannas


Para crear ventanas hay dos pasos que hay que hacer antes,primero se tiene que crear una clase, y es en ese momento donde se le asocia la funcion XXX de reacción. Luego de haber creado la clase, ya puedes crear ventanas donde la asocias con la clase mediante una variable que creo que era un string.
